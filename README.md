v 1.0.0
介绍

    这是用 node + mongodb + bootstrap编写的中大型的一个 web blog 
    普通用户在注册然后登陆之后是可以查看全文并且评论的。
    管理员可以进入后台管理查看用户信息，查看和操作类别信息, 查看和操作内容信息
查看注意、、

    依赖node，自己的环境是 node 
    依赖mongodb储存用户数据，需要自己创建个db文件夹存储数据然后连接数据库才能打开网站！！！
